# Sorteo PlayStation - React (Vite)

Proyecto convertido a React con Vite.

Requisitos:

- Node.js >= 16

Instalación y uso:

```bash
npm install
npm run dev
```

- Abre `http://localhost:5173` (Vite) para ver la app.
- La imagen de fondo está en `public/bg.svg` y se posiciona en el fondo de la página.

Si prefieres que empuje los cambios a un fork remoto, dime tu usuario de GitHub y lo configuro.
